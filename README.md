# My Shopping List
## About
A simple webpage which display an unordered list of items. New item name and it's quantity can be added in the list. Deletion of items can also be done. 
## Technologies Used
* React Js
* Bootstrap
 ## How To run project
 
1. Run this NPM command to install NPM dependencies.
```
  npm install

```


2. To run
 ```
 npm start
 
 ```
